Typora 美化

修改GIthub主题的css样式，在 Typora 的偏好设置中，点击 -》外观 -》 打开主题文件夹，在文件夹中选择**github.css**，下面修改包括：

  • 字体：手写字体==华康手札体 W5P==
  • 高亮颜色
  • 表格优化
  • 复选框优化

（Windows）安装字体：华康手札体 W5P

  • 将解压出来的华康手札体W5P.ttf，复制到**"C:\Windows\Fonts"**中，复制及自动安装，然后在 Typora 中，将在安装字体的名字复制，写入到主题的 css 样式中。
